Pathname

A pathname is the complete address or location of a file or directory in a file system. It is used to access files and directories in operating systems like UNIX.

Types of Pathnames

  • Absolute Path: The full path starting from the root directory (e.g., `/home/user/docs/file.txt`).
  • Relative Path: The path relative to the current working directory. Learn more
